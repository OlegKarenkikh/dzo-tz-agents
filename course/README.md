## Для кого этот курс?

- Вы первый раз открываете терминал
- Вы хотите понять, что такое LLM, агенты, инструменты и API
- Вы хотите запустить агентов ДЗО/ТЗ и не знаете с чего начать

## Как учиться?

1. Читайте уроки по порядку — каждый следующий опирается на предыдущий
2. Выполняйте команды прямо в терминале
3. Не бойтесь ошибок — в уроке 2 вы узнаете, что баги это нормально
4. Все иллюстрации находятся в папке [images/](images/)

> 💡 **Совет:** Открывайте [README.md](../README.md) проекта рядом с уроком.
> Там вы найдёте все команды для быстрого старта.
