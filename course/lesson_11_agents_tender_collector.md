## 📌 Агент Тендер (agent21)

### Назначение

**Агент Тендер** — вспомогательный агент для анализа тендерной документации.
Извлекает список документов, которые требуются от участников тендера.

> **Нумерация `agent21`:** цифра «21» означает «агент 2, подчинённый агенту 1».
> Это вспомогательный (peer) агент для Агента ДЗО, не самостоятельный.

### Файлы

```
agent21_tender_inspector/
├── agent.py      ← ReAct-агент
├── runner.py     ← обработчик входящих документов
├── tools.py      ← инструменты парсинга
└── ALGORITHM.md  ← бизнес-правила
```

### Что умеет

- Разбирает PDF и Word-файлы тендерной документации
- Извлекает требуемые документы (анкеты, сертификаты, гарантии)
- Возвращает структурированный список требований

### Практика: вызов через curl

```bash
curl -s -X POST http://localhost:8000/api/v1/tender/inspect \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "document": "ТЕНДЕРНАЯ ДОКУМЕНТАЦИЯ\n\nТребуемые документы:\n1. Анкета участника\n2. Выписка из ЕГРЮЛ\n3. Справка об отсутствии задолженностей\n4. Банковская гарантия"
  }' | python3 -m json.tool
```

## 📦 Агент Collector (agent3)

### Назначение

**Агент Collector** — сборщик анкет участников тендерного отбора.
Получает письма от потенциальных участников, извлекает вложения (анкеты в PDF/Word/Excel)
и проверяет их комплектность против эталонного шаблона.

### Файлы

```
agent3_collector_inspector/
├── agent.py      ← ReAct-агент
├── runner.py     ← IMAP-опросчик почты
├── tools.py      ← инструменты проверки анкет
└── ALGORITHM.md  ← бизнес-правила
```

### Что умеет

- Читает входящие письма от участников тендера
- Извлекает вложения (PDF, DOCX, XLSX)
- Сверяет каждую анкету с эталонной формой
- Формирует сводный отчёт: кто прислал полный пакет, кто нет

### Практика: запуск Collector

```bash
# Отправить анкету на проверку
curl -s -X POST http://localhost:8000/api/v1/collector/inspect \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "document": "АНКЕТА УЧАСТНИКА\nНазвание компании: ООО Ромашка\nИНН: 1234567890\nКонтактное лицо: Иванов И.И.\nТелефон: +7-900-000-00-00"
  }' | python3 -m json.tool
```

## 🔄 Полная картина: как агенты работают вместе

```
📧 Входящее письмо от ДЗО
         ↓
    Агент ДЗО (agent1)
    ├── Есть ТЗ? → Агент ТЗ (agent2) ← анализирует разделы
    ├── Есть тендер? → Агент Тендер (agent21) ← парсит документы
    └── Итог: принять / доработать / эскалировать

📧 Письма от участников тендера
         ↓
    Агент Collector (agent3)
    └── Сверяет анкеты с шаблоном → отчёт о комплектности
```
